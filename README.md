# How to use

| PARAMS | DESCRIPTION          | DEFAULT VALUE               |
|--------|----------------------|-----------------------------|
| tc     | Text color (hex)     | 000000                      |
| ts     | Text size            | 12                          |
| tp     | Text padding         | 10                          |
| sc     | Stroke color         | 000000                      |
| sw     | Stroke weight        | 2                           |
| dc     | Distance from center | 80                          |
| data   | The data to display  | python,php,javscript,nodejs |
